Website

This website is built using zola. The theme is no-style-please. The fonts are iosevka variants. The colors are from the nord palette.

On push to master to this repository, github actions trigger a zola build and push to the gh-pages branch. That branch name is special, because it then triggers a github pages deploy action to this website.

The iosevka font is trimmed down to about 12Kb using python font tools (inspired by Xe).

pyftsubset ./IosevkaAile-Regular.ttf \
--output-file=iosevkaaile-regular.woff2 \
--flavor=woff2 --no-hinting --desubroutinize \
--unicodes="U+20,U+21,U+23,U+27-29,U+2B-3A,U+40-50,U+52-57,U+59,U+61-7A,U+F6"
